Engineering Challenges Solved

🔐

Custom Authentication System

Built a complete user authentication system independent of WordPress, including token-based sessions, device fingerprinting, multi-device management with approval workflows, and QR-code-based login for kiosk environments.

⚡

Real-Time Multi-Device Sync

Implemented WebSocket-based real-time synchronization between POS terminals, admin dashboards, and customer devices. QR scans appear instantly across all connected clients with suspicious activity detection and alerts.

🤖

Context-Aware AI Support

Integrated Claude API with dynamic system prompts that know the entire platform. The AI detects which page the user is on, which store they belong to, and provides relevant answers. Includes escalation flow to human support when needed.

🌍

Multi-Language Architecture

Designed a translation system supporting 5 languages (German, Hungarian, Romanian, English, Italian) with 2,000+ translation keys each. Includes context-specific translations for main app and repair subsystem.

🛡

Anti-Fraud & Security

Built device fingerprinting to prevent QR code abuse, with admin approval workflows for new devices. Implemented nonce validation, CORS handling, rate limiting, and cross-domain API security for external integrations.

📱

Cross-Platform Delivery

Shipped as a responsive PWA for web, native iOS app via Swift with WebView bridge, and Fully Kiosk-compatible for tablet POS. Set up automated CI/CD with Codemagic for TestFlight deployment on every build.
